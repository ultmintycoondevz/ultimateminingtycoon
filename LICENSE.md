## 🔧 When Things Go Wrong

### Common Issues and Solutions

#### Issue: Script Won't Launch

**Symptoms:** Application crashes immediately or doesn't open

**Solutions:**
1. Verify system meets minimum requirements
2. Run as administrator (Windows) or grant permissions (macOS)
3. Disable antivirus temporarily and add to exclusions
4. Reinstall with fresh download
5. Check Windows Event Viewer for error codes

#### Issue: Features Not Working In-Game

**Symptoms:** Overlay doesn't appear or functions are inactive

**Solutions:**
1. Confirm you're using supported popular titles
2. Verify hotkey isn't conflicting with other software
3. Check if game was updated (wait for script update)
4. Restart both the game and the script
5. Review configuration file for syntax errors

#### Issue: High CPU/Memory Usage

**Symptoms:** Computer becomes slow or fans run loudly

**Solutions:**
1. Reduce `max_threads` value in configuration
2. Set CPU priority to "medium" instead of "high"
3. Decrease `memory_limit` parameter
4. Close unnecessary background applications
5. Enable "battery saver" mode on laptops

#### Issue: Mobile App Crashes

**Symptoms:** Application force closes on Android/iOS

**Solutions:**
1. Clear app cache and data
2. Ensure 2GB+ RAM available on device
3. Update to latest mobile OS version
4. Reinstall application with fresh download
5. Disable battery optimization for the app

#### Issue: Configuration Not Saving

**Symptoms:** Settings reset after closing application

**Solutions:**
1. Check file permissions in installation folder
2. Don't run from temporary directories
3. Ensure disk has sufficient free space
4. Run with elevated privileges
5. Manually backup config.yml file
